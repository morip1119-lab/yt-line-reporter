3 Tips
3つのコツの活用
1
既存のAPIを組み合わせる
YouTube Data API・LINE Messaging API という既存サービスを活用し、自分で作るのはデータの「繋ぎ役」だけにした。ゼロから作らず、あるものを組み合わせることで短期間で完成できた。
2
サーバーレス（GitHub Actions）で運用コストゼロ
自前サーバーを持たずに GitHub Actions の無料枠で毎日自動実行。PCの電源が切れていても動く。インフラ管理不要で運用が続けやすい。
3
AIと一緒に段階的に作る
完成形を一度に作ろうとせず、「まず1チャンネル送れる」→「デザイン改善」→「複数チャンネル対応」→「自動化」と段階的に拡張。Cursorと対話しながら詰まった部分をその場で解決した。
